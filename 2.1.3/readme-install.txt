Installationsanweisungen für den drawWEBKATALOG
drawWebkatalog Software Script Install

Hier finden Sie die Installationsanweisungen für die Installation der Webkatalog Software drawWEBAKTALOG. In den folgenden Punkten ist Schritt für Schritt erklärt wie Sie bei dier Installation vorgehen. Sollten Sie jedoch Probleme haben Helfen wir Ihnnen gerne.


1. Schritt
In der Datei /configs/config.inc.php öffnen und die MySql Benutzerdaten an Ihren Server anpassen.
Die Datai /.htaccess anpassen und die Kommentierung(#) entfernen.

2. Schritt
Alle Daten auf den Ihren FTP-Server hochladen.

3.Schritt
Danach das Installationsverzeichnis aufrufen "http://www.deinedomain.de/install/". Überprüfen Sie Ihre Daten nocheinmal, wenn alle Daten korrekt eingeben wurden. Legen Sie noch einen Benutzernamen und ein Passwort fest, mit denen Sich in den Adminbereich einlogen können. Anschließend klicken Sie auf "Webkatalog installieren" und die Installtion beginnt.

4. Schritt
Bei der Installtion wurden folgende Tabellen mit Daten in Ihre Datenbank geschrieben.

- Adminbenutzer

    * name
    * Passwort

 

- Settings

    * url
    * name
    * genre
    * email

 

- Pages ( Standard Seiten einfügen )

    * Startseite
    * Seite anmelden
    * Top Seiten
    * Neusten Seiten
    * Suchen
    * Hilfe
    * Kontakt
    * Impressum
    * Nutzungsbedingung
    * AGBs

 

5. Schritt
Die Installation ist damit beendent und das Installationsverzeichnis können Sie nun von Ihrem Server löschen.
